## Latency

- Discovery p95 ≤ 180s
- Per-agent p95 ≤ 8s
- Render p95 ≤ 8s

## Availability

- Gateway availability ≥ 99.9%
- Orchestrator availability ≥ 99.5%

## Error Budgets

- 0.5% monthly for orchestrator; 0.1% gateway

## Cost SLO

- Standard artefact ≤ $0.75; Premium ≤ $2.00
