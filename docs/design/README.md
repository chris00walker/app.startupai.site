# Design Documentation

This folder contains permanent design specifications and guidelines for the StartupAI platform.

## Contents

### Root Files

| File | Purpose |
|------|---------|
| `figma-design-system-spec.md` | Design system specification for Figma |
| `figma-workflow-guide.md` | Guide for design agents working with Figma |
| `404-og-image-spec.md` | Specification for 404 page OG image |
| `brand-review-404-compass.md` | Brand review for 404 Compass design |

### Subfolders

| Folder | Purpose |
|--------|---------|
| `component-specs/` | Component-level specifications |
| `ux-specs/` | User experience specifications |
